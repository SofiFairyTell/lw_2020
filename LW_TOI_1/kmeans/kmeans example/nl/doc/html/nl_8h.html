Подробное описание

Файл подключает все остальные заголовочные файлы библиотеки.

Для компиляции вашей программы с использованием настоящей библиотеки достаточно подключить только данный заголовочный файл. Для сборки (линковки) необходимо подсоединить объектные (или библиотечные) файлы библиотеки.

Необходимо сделать:
Во всех модулях минимизировать число вызовов malloc, create и т.п. Передавать в качестве параметров функций рабочие массивы.
Необходимо сделать:
Во всех модулях в описаниях *a называть не массивом, а указателем, указывающим на область памяти нужной лины (входной параметр).
